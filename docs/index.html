1. Background

  First, we will discuss the Planar and Toroidal embeddings of a graph in order to establish the vital features to the composition of an embedding algorithm.

1a) Planar Graphs [1: pg 12] [3: Section 9.1]

  A Planar Graph is a graph $G$ which can be drawn on a plane without any crossing edges. By Kuratowski's Theorem, it is known that a graph $G$ is planar if and only if it does not contain any subgraphs that are a subdivision of $K_5$ or $K_{3,3}$.
